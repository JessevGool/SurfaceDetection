\doxysubsubsection{\texorpdfstring{startScan()}{startScan()}}
{\footnotesize\ttfamily void Ray\+Test.\+start\+Scan (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

This method initializes the scan. The scan size is based on the size of the parent gameobject. The scan is subdivided into a grid based on the grid\+Size variable. The resolution of the scan is based on the scan\+Resolution variable. Both of the variables named above can be set in the settings panel in the GUI.
