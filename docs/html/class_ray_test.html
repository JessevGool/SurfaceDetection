◆ SwitchDrawHits()

void RayTest.SwitchDrawHits ( )

This method initializes the scan. The scan size is based on the size of the parent gameobject. The scan is subdivided into a grid based on the gridSize variable. The resolution of the scan is based on the scanResolution variable. Both of the variables named above can be set in the settings panel in the GUI.
